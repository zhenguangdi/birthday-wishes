# Birthday Greeting Web Project

A beautiful static birthday greeting web project featuring multiple display pages and animation effects.

## Project Overview

This project is a birthday greeting web application built with HTML, CSS, and JavaScript, offering rich visual effects including animated birthday cakes, memory timelines, greeting pages, and more.

## Page Features

### 1. Birthday Cake Page (BirthdayCake.html)
- Dynamic birthday cake animation
- Candle flame effect
- Birthday greeting text
- Interactive buttons

### 2. Memory Timeline (Memories.html)
- Full-page scrolling effect powered by fullPage.js
- Timeline-based display of memories
- Combined image and text presentation
- Bubble background animation

### 3. Login Page (login.html)
- Form-based login interface
- Dynamic bubble background effect
- Input field animations

### 4. Easter Egg Page (EasterEgg.html)
- Hidden surprise content

### 5. Home Page (index1.html)
- Tab switching functionality
- Command menu interface

## Technology Stack

- **Frontend Framework**: jQuery
- **Page Scrolling**: fullPage.js
- **Styling**: CSS3 animations, Flexbox layout
- **Responsive Design**: Supports various screen sizes

## Usage Instructions

1. Open `html/BirthdayCake.html` directly in your browser to view the birthday cake animation.
2. Open `html/Memories.html` to view the memory timeline.
3. Open `html/login.html` to view the login page effect.

## Customization

- Replace images in the `img/` directory with your own content.
- Replace audio files in the `music/` directory.
- Modify the greeting text within the HTML files.

## Browser Support

- Chrome
- Firefox
- Safari
- Edge

The project includes a `vercel.json` configuration file for direct deployment to the Vercel platform.
